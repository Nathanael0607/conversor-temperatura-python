# Conversor de Graus em Python 🌡️

Este é um projeto simples desenvolvido em Python para realizar a conversão de temperaturas entre as escalas Celsius e Fahrenheit. O objetivo principal deste projeto foi praticar conceitos fundamentais da linguagem, como funções, estruturas de repetição (`while`), condicionais (`if/elif/else`) e tratamento de entrada de dados.

## 🚀 Funcionalidades

- Converte temperaturas de **Celsius para Fahrenheit**.
- Converte temperaturas de **Fahrenheit para Celsius**.
- Interface amigável via console com menu de opções.
- Encerramento controlado do programa.

## 🛠️ Tecnologias Utilizadas

- **Python 3**: Linguagem principal utilizada.
- **Biblioteca `sys`**: Utilizada para o encerramento seguro do sistema após a execução.
